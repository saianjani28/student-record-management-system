# student-record-management-system
Student Record Management System is a web based application that stores, manages, and tracks student academic and personal data digitally. It allows admins to add, update, search, and delete student records, manage attendance and grades, and ensure secure, fast, and paperless record handling for institutions. It improves efficiency and data result.
Features
Role-based login with Admin, Staff, User, and Guest access levels
Add new students with unique roll number validation
Display all student records in a formatted list
Search for students by roll number
Update student name or marks
Delete student records safely using temporary files
Supports full names with spaces
Persistent data storage using text files with automatic backups
Role-Based Login
The program reads credentials from credentials.txt. Supported roles include:

ADMIN: Add, View, Search, Update, Delete
STAFF: View, Search, Update
USER: View, Search
GUEST: View only
Student Record Operations
Add a new student with unique roll number enforcement
Display all students
Search for a student by roll number
Update existing student information
Delete a student record
Supports full names with spaces
